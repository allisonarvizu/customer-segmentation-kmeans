# Customer Segmentation using K-Means (Unsupervised Learning)

## Overview
This project applies K-Means clustering to segment customers based on annual income and spending behavior.

## Techniques Used
- Data generation (synthetic dataset)
- Feature scaling (StandardScaler)
- Elbow method for optimal cluster selection
- K-Means clustering
- Data visualization (Matplotlib)

## Key Insights
The model identified distinct customer segments such as:
- High-value customers (high income, high spending)
- Underutilized customers (high income, low spending)
- Low-value customers (low income, low spending)

These insights can help businesses improve targeting and retention strategies.
